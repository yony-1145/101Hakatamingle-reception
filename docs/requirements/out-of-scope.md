# MVPスコープ外・未決定事項

## MVPスコープ外

以下はMVPでは対応しない。フェーズ2以降で検討する。

- データエクスポート（参加者リスト等のCSV出力）
- ポイント / ランク制度（ゴールド・ファミリー等）
- クーポン配布
- SNSログイン（Google / LINE）
- プッシュ通知（PWA / ネイティブ）
- 分析ダッシュボード
- 福岡マップ / SNS機能
- キャリア相談 / 留学相談予約
- ネイティブアプリ化
- 多言語対応（3言語以上への拡張）
- アプリ内通知
- **ゲスト→会員アカウント引き継ぎ**（同メアドで会員登録時に過去ゲスト履歴を自動紐付け / Stripe Customer引き継ぎ含む）
- **会員からの+1ゲスト同伴枠**（会員が代理で事前にゲスト枠を確保する機能 / ゲスト本人のメアド/認証不要）
- **ゲスト事前登録**（イベント詳細ページからの事前ゲスト登録 / MVPは会場QR経由の当日登録のみ）
- **ゲストマイページ**（OTP再ログインによる過去履歴閲覧・キャンセル等の永続的なゲストUI）

## 未決定事項

- メール配信サービスの選定（Resend / SendGrid 等）
- コスト試算（Supabase / Stripe 手数料 / メール配信）
- STUDIO側のLPからアプリへの導線設計
